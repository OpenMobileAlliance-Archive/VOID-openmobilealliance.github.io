OMNA Vendor/Operator ID Registry

OMNA maintains this registry to record values used by known services when invoking the OMA-defined LTE Positioning Protocol extensions (LPPe).

The OMA-LPPe-VendorOrOperatorID is used to identify the vendor/operator using the proprietary data content. Two methods are provided. The first is a method, in which the vendor/operator identifier is standardized. The alternative method is not to use the standardized vendor/operator identifier, but the generalized method based on the CRC sum of the vendor/operator name.

The standard-VendorOrOperatorID element (which is part of OMA-LPPe-VendorOrOperatorID and which applies to the first method) is defined in the LPPe Enabler Release.

In order to register an LPPe Vendor ID, please use the request form which may be found here.
