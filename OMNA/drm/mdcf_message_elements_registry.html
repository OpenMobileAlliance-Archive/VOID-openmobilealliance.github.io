OMA MPEG-2 DRM Control Format (MDCF) Message Elements Registry

The MDCF protocol ( OMA-TS-SCE_MDCF-V1_0 ) includes several message elements which permit values to be assigned by registration. This OMNA registry supports the following message elements:

  • CA_descriptor_message_ID
  • CID_message_ID
  • rights_URL_message_ID
CA_descriptor_message_ID registry
This registry records assigned values for the CA_descriptor_message_ID field as described in section 5.3.1 of OMA-TS-SCE_MDCF-V1_0 identifies a specific message.

The CA_descriptor_message_ID field is eight bits long and the possible values have been segregated as follows:

Range Description
0x00-0x1F reserved for use by OMA
0x20-0xF7 values used for MDCF applications assigned in this registry
0xF8-0xFE available for private/experimental use - not recorded
0xFF reserved for future use
