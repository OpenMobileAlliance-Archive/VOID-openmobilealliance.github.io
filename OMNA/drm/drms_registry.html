OMA Digital Rights Management System (DRMS) Registry

OMNA maintains a registry of Digital Rights Management System (DRMS) URNs needed to fulfill the labelling requirements for system names. The System name element is described in the DRM Rights Expression Language (REL) specification (OMA-TS-DRM-REL-V2_1).

DRMS Naming Model

The DRMS Identifier registry namespace is divided into three:

  • OMA-defined registrations
  • External organizations registrations
  • Private registrations

These labels are defined in the following table:

Label Description
Name Model Description
oma-<type> OMA-defined Registrations
org-<OrgLabel> External Organization Registrations
x-<PrivateLabel> Experimental or Private Uses (Not Registered)

All registration requests must follow the guidelines for DRMS Registrations.
OMA-Defined DRMS System Registry

Registration requests to this list must be approved by the DRM Working Group. The format of the OMA-defined registrations has the form:

urn:oma:drms:oma-<Type>:<SystemName>

where: <Type> characterizes the DRMS system form
     <SystemName> declares a specific system instance

Each entry provides information related to the group registering the system as well as a description of its usage.

External Organization Registry

Registrations in this list are of root URNs assigned to external organizations. As roots, they may have subordinate components defined by the registering entity. OMNA does not track these items and does not make any claims as to appropriate use for the resulting values. Please examine the links to get information for the usage as defined by the specific organizations.
