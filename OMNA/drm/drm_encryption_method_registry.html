OMA DRM EncryptionMethod Registry

OMNA maintains this registry to record values used for the EncryptionMethod field for the DRM Control Format (DCF) as defined by the OMA-defined DRM enabler.

The EncryptionMethod field element is defined in the OMA-TS-DRM_DCF-V2_0 specification. It is used to select a particular method to decrypt the associated content.

New entries to be added to this table must be reviewed with the OMA DRM WG. They will assure that the required data is correctly structured and does not conflict with existing service descriptions.

The EncryptionMethod field is eight bits long and the possible values have been segregated as follows:

Range Description
0x00-0x1F reserved for use by OMA
0x20-0xF7 values available for non-OMA registrants
0xF8-0xFE available for private/experimental use - not recorded
0xFF reserved for future use
