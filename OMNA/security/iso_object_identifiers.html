OMA ISO Object Identifiers

OMNA manages the fields and values assigned to the tree defined by this Object Identifier (OID) root.

An ASN.1 format file with the identifiers can be loaded at OMA-OIDs.asn

The information to be included in a new OID request may be found here. At this time, only requests from other OMA groups are being considered.

The root of the OMA OID table has been inherited from the registration made by the WAP Forum. An alias "oma" has been registered as well, such that "wap" and "oma" can be used interchangeably to address the same OID space. This root is defined as:

wap OBJECT IDENTIFIER ::= {joint-iso-itu-t(2)
international-organizations(23)
wap(43)[other identifier: oma]}
