OMA GBA Protocol Identifiers

The following registry records OMA-declared Protocol Identifiers in support of Generic Bootstrap Architecture (GBA). These Protocol Identifiers are defined as the Ua Security Protocol Identifier [ 3GPP TS33.220 ] as a five octet string. The first byte is the Organization Octet followed by four octets which unambiguously declare the specific protocol scheme to utilize.

OMA has received the value of 0x03 for the Organization Octet to permit it to declare protocol identifiers in its domain. This registry serves to record all such assignments.

Allocation Scheme for OMA-defined GBA Protocol Identifiers

OMA will allocate the GBA Protocol Identifier as the five octet string 0x03,0xWW,0xXX,0xYY,0xZZ. These strings will cover different types of code schemes according to the following table:

Allocation Type Pattern Rule Description
Non-versioned Schemes WW=0x00 & XX=0x00 Schemes explicitly identified by the YY and ZZ values
Versioned Schemes WW=0x00 & 0x01<=XX<=0xFF Schemes where the YY and ZZ values distinguish specific options
Reserved 0x01<=WW<=0xFE Intended to provide for future uses
Experimental WW=0xFF Uses will not be registered

Acquiring an OMNA-assigned GBA Protocol Identifier

At present, OMNA assignments into this registry are made only for OMA Working Groups.
