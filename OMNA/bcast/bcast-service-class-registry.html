OMNA BCAST Service Class Registry

OMNA maintains this registry for Service Classes associated with BCAST services.

The Service Class names follow a simple naming model that permits unambiguous usage of the names. Each Service Class name is structured as a URN to assure uniqueness. The model follows the following grammar:

{ClassName} ::= { {OMAname} | {ExtName} }

{OMAname}   ::= urn:oma:bcast:oma_bsc: { {ClassLabel}:{Version}}

{ExtName}   ::= urn:oma:bcast:ext_bsc_ {Org}:{ClassLabel}:{Version}}


Where

- {ClassLabel}is a string made up of lowercase alpha characters
- {Version}    is a string consisting of numbers and periods (e.g. '1.2')
- {Org}     is a string that conveys the name of the external organization

Prefix Registry for External Organizational Broadcast Service Classes

OMNA does not register externally defined broadcast service classes. It does, however, provide a URN prefix registry which provides a means to assign unique labels for their service class names.

Organizations seeking a prefix assignment should submit an initial request. The form for submitting a request may be found here.
