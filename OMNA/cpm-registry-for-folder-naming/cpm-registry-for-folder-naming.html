CPM REGISTRY FOR FOLDER NAMING
The non-CPM folders are allowed to be specified inside the CPM Mesage Store [defined in OMA-TS-CPM_Message_Store-V2_1], by any external enabler, service or applications .These top-level folders must have unique names in order to avoid any conflicts. New entries to be added to this table must be reviewed with the OMA COM WG. They will assure that the required data is correctly structured and does not conflict with existing folder names and definitions.
