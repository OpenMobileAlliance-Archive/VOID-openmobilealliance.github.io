OMNA XCAP Application Unique ID (AUID) Registry
OMNA maintains this registry to record Application Unique IDs (AUIDs) associated with XCAP resources defined in OMA Enablers using the OMA namespace. As such, this registry is only available to OMA groups and no support for external registrations is offered.

The XCAP AUID records are developed as vendor-based IDs as referenced in RFC 4825. The vendor domain part covered by this registry pertains to 'org.openmobilealliance' which is the reversed name string for the recognized OMA domain.

It should be noted that OMA Enablers may also utilize XCAP AUIDs which are declared in standards track RFCs. These are recorded by IANA and are available at the IANA XCAP Parameters page.

More details on registration can be found on OMA Portal at http://member.openmobilealliance.org/ftp/rel/gen_info/OMNAprocs_XCAP_AUID.htm.
