OMA WMLScript Library IDs
OMNA assigns numbers for registered WMLScript libraries. The number space is divided into three segments:
Segments
Range
Description
0x00 to 0x7F
Well-Known values are set as specified by the specification group processes.
0x80 to 0x3FFF
Reserved values, registered and managed by OMNA.
0x4000 to 0x1FFFFF
Experimental values, registered by OMNA only for advisory purposes.

In order to register a WMLScript library identifier, a short description of its use must be provided. It is helpful if this description includes the list of funciton IDs that the library uses. The information to be included in a request may be found here.

See WMLScript 30-April-1998[WMLS] (PDF) and WMLScriptStandard and Libraries 30-April-1998 [WMLSLib](PDF) for more information on library identifiers.
