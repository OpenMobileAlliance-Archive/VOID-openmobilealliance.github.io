OMA WBXML Public Document IDs
OMNA assigns numbers for registered public document IDs. The number space is divided into three segments:
Segments
Range Description
0x00 to 0x7F Well-Known values are set as specified by the OMA MAG/MAE group processes.
0x80 to 0x3FFF Reserved values, registered and managed by OMNA.
0x4000 to 0x1FFFFF Experimental values, registered by OMNA only for advisory purposes
In order to register an WBXML public document identifer number, a short description the DTD and the public DTD name must be provided. The information to be included in a request may be found here.

See WAP-192-WBXML for more information on public document identifier use in WBXML.
