OMNA PUSH Message Header Code Registry
OMNA maintains a registry of field codes that are assigned to OMA Push message headers. OMA Working Groups and other SDOs (Standard-Developing Organizations) can register OMA Push message extension headers in this registry. The assignment of field codes to the OMA Push message headers is done by OMNA.
A form for submitting a request may be found here

Extension header names conform to the following structure:

FullHdrName = 'X-Oma-' [SDOName '-'] HdrName ; OMA is the default SDO name

SDOName = 1*VCHAR

HdrName = 1*VCHAR

; (RFC 5234 defines VCHAR as any visible character

; i.e. any character with ASCII value in the range

; 0x21 through 0x7E)
