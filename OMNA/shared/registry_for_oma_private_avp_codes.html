OMA Registry for OMA Private AVP Codes

This registry is used to record Attribute Value Pair (AVP) Codes in the OMA address space. These OMNA registered AVP Codes are assigned to OMA Working Groups on a first come, first served basis. Reference to the defining specification for each defined attribute is provided. The AVP Codes used on this page are associated with Diameter system usage as defined in RFC3588. The AVP code header must include the V (Vendor-Specific) bit which flags the presence of the optional Vendor-Id field which would be set to the OMA Private Enterprise Number value of 30079.
