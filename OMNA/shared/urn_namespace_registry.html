OMA (URN) / Namespace Registry

OMA maintains a namespace registry based on URNs in support of the specifications produced by OMA. The OMA registry allocates these URNs based upon a reserved NID ('oma'). See RFC 4358 for the document which reserved this NID.

The Namespace Specific String (NSS) of all URNs that use the 'oma' NID will have the following structure:

urn:oma:{OMAresource}{ResourceSpecificString}

For each OMAresource there is room for further allocations specific to the resource. The following table links to the current URN allocations using this mode.
