OMNA Registry for OMA Smartcard Application (AID) Codes

This registry is used to record Smartcard-based Application Id (AID) Codes in the address space allocated OMA per ISO/IEC 7816-5.

The AID identifier is divided in two key elements - a 5-byte Registered application provider Id (RID) and a 7 to 11 byte Proprietary application identifier extension (PIX). For this registry, the AID is set per the assignment of the RID to the value assigned to OMA (0xA000000412) and the PIX code will be further divided into a 2 byte OMA application code followed by a 5 to 9 byte value that is managed by the assigned application.
