OMNA BCAST Parental Rating System Registry

This registry provides values associated with the rating_type and rating_value fields used by the OMA BCAST Enabler. The rating system fields provide a two tier scheme by which a particular rating domain is selected and the actual rating codes are determined for that domain. The rating_type field is used to select the rating domain and the rating_value selects the specific rating within that domain.

The rating system scheme as recorded in this registry is described in OMA-TS-BCAST_Service_Guide-V1_0 and OMA-TS-BCAST_SvcCntProtection-V1_0.

Some schemes are in the table for legacy reasons. These are marked as (deprecated). It is not advised to use these in implementations unless required. OMNA reserves its right to change its policy on support for these legacy code sets in the future.

The OMA BCAST Working Group will review requests for assignments and will provide numeric assignments of the values for the rating_value fields where needed. Requests may be made using this request form or via liaison sent to OMA-LIAISON.

Follow-ups will be made to collect any missing information. Such information typically includes a description of the parental rating scheme for which registration is seeked. Furthermore, the requester may indicate the set of preferred rating_values to be assigned. Note that OMNA reserves the right to actually assign values differing from those.
